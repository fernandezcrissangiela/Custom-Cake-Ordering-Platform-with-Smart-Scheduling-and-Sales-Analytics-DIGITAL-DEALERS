The purpose of this project is to create an easy-to-use ordering website for C and N Bakery to replace their manual paperwork and messy chat logs. For customers, the site offers a simple menu with budget price filters, an interactive custom cake builder that calculates prices automatically, a reference photo uploader, and flexible checkout options like Bank transfer, GCash QR or Cash on Delivery. For the owner and staff, the site provides a secure admin dashboard featuring simple order and sales summaries, a 6-stage order tracking((Pending, Confirmed, Baking, Ready, Completed, Cancelled ), an interactive pop-up window to review design blueprints, a quick on-and-off menu stock switch, an order calendar to prevent overbooking, and visual sales charts that automatically track monthly revenue.
